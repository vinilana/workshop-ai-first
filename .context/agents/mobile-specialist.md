## Not Applicable

DOM Pagamentos is a web-only project. There are no mobile applications, no React Native packages, and no plans for mobile in the workshop scope.

The two frontends (`packages/frontend-v1` and `packages/frontend-v2`) are browser-based React apps served by Vite.

If mobile support were ever added, the API Gateway at port 3000 provides the logical integration point — a mobile client would target the same endpoints as the web frontend.

For now, redirect mobile-related questions to the Frontend Specialist playbook.
